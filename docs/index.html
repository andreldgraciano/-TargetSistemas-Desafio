Laço de Repetição

Observe o trecho de código abaixo e informe qual será o valor final de SOMA.

Código

int INDICE = 12, SOMA = 0, K = 1;
enquanto K < INDICE faça{
   K = K + 1;
   SOMA = SOMA + K;
}
imprimir(SOMA)

Demonstração de Iteração 

int INDICE = 12, SOMA = 0, K = 1;
enquanto K < 12 faça{
   K = K + 1;
   SOMA = SOMA + K;
}
imprimir(SOMA)
